# Security

## Reporting

If you believe you found a security issue, please avoid posting sensitive exploit details in a public issue.

Preferred path:

1. use GitHub security reporting for this repository if available
2. if private reporting is not available, open a minimal public issue that states a security concern exists without posting secrets, tokens, or exploit instructions

## Scope

This repository is a runtime kit and public distribution package.
Please report issues that affect:

- runtime entrypoints
- installation and deployment safety
- credential handling
- unsafe defaults
- connector exposure

## Do Not Post

- live credentials
- private keys
- access tokens
- private infrastructure details
