# Contributing

Thanks for taking an interest in improving the runtime kit.

## Good First Contribution Areas

- public documentation clarity
- installation and validation guidance
- portability improvements
- small runtime-tool fixes
- public issue reporting and maintenance hygiene

## Before Large Changes

Please open an issue first if you want to propose:

- a new integration surface
- a significant runtime workflow change
- a major public scope expansion

## Important Project Note

This public repo is refreshed from a maintained private canonical source of truth.
That means accepted public changes may be ported back into the canonical source and then re-exported here, rather than merged only as an isolated public-only patch.

## Contribution Expectations

- keep changes focused
- prefer clear docs and reproducible behavior over extra framework layers
- do not add secrets or private-origin material
- keep the public entry layer understandable to external technically literate users
